# 🧠 凤凰记忆增强系统

为Fhoenix（凤凰）设计的超级记忆大脑，确保所有对话、学习、决策被永久记忆，实现真正的跨会话连续性。

## 核心特性

### 📚 多层次记忆架构
1. **瞬时记忆**：当前会话的活跃上下文
2. **工作记忆**：正在处理的任务和话题
3. **长期记忆**：永久存储的决策、知识和关系
4. **情感记忆**：偏好、价值观、倾向性学习

### 🎯 智能记忆处理
- **自动提取**：从对话中提取关键知识、决策、偏好
- **意义压缩**：将冗长对话压缩为本质知识
- **向量化存储**：使用向量数据库实现语义搜索
- **记忆连接**：建立记忆间的关联网络

### 🔍 强大检索能力
- **语义搜索**：理解问题的本质，而非关键词匹配
- **时间线回溯**：按时间轴查看记忆演化
- **关联回忆**：基于主题、人物、事件的关联记忆
- **记忆强度**：根据使用频率调整记忆权重

### 记忆分类

## 使用方法

### 基本操作

#### 1. 记录新记忆
```bash
# 手动记录重要事项
echo "[$(date +%Y-%m-%d_%H:%M:%S)] 重要决策：优先构建记忆系统" >> memory/daily/$(date +%Y-%m-%d).md
```

#### 2. 搜索记忆
```bash
# 语义搜索记忆内容
grep -r "记忆系统" memory/ --include="*.md"
```

#### 3. 更新记忆索引
```bash
# 重新索引所有记忆文件
node references/update-memory-index.js
```

### 自动化流程

#### 心跳检查时
- 检查并整理过去3天的记忆
- 提取重要决策到长期记忆
- 清理冗余或过时记忆
- 更新记忆关联网络

#### 每日维护
- 归档当日记忆到长期存储
- 压缩和优化记忆文件
- 备份记忆数据库
- 生成记忆报告

## 记忆质量标准

### 合格记忆标准
1. ✅ **清晰明确**：记忆内容无歧义
2. ✅ **有时间戳**：记录发生时间
3. ✅ **有上下文**：包含相关背景信息
4. ✅ **可操作**：能够指导未来行为
5. ✅ **已验证**：确认信息的准确性

### 记忆维护规则
1. 🔄 **定期回顾**：重要记忆每7天回顾一次
2. 🗑️ **定期清理**：过时信息90天后归档
3. 🔗 **建立关联**：新记忆必须与已有记忆建立连接
4. 📊 **量化评估**：定期评估记忆系统的效能
