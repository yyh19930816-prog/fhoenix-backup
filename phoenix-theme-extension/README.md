# 🚀 凤凰主题 Chrome 扩展

为 OpenClaw 控制界面注入华丽的凤凰主题视觉系统

## ✨ 功能特性

- 🔥 **凤凰火焰色彩方案** - 红→橙→金的渐变过渡
- 🎨 **非侵入式CSS注入** - 不影响原有功能的安全覆盖
- ⚡ **实时主题切换** - 一键启用/禁用主题
- 📱 **响应式设计** - 适配桌面和移动端
- 🔧 **可定制设置** - 主题变体、动画级别选择
- 🌟 **流畅动画** - 现代化的交互效果

## 🛠️ 安装方法

### 开发者模式安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本扩展的文件夹 (`phoenix-theme-extension`)
5. 扩展安装完成，工具栏会出现凤凰图标

### 使用方法

1. 点击工具栏的凤凰图标打开控制面板
2. 切换"启用主题"开关来激活/禁用主题
3. 根据需要调整主题变体和动画设置
4. 刷新页面查看主题效果

## 🎯 兼容性

- **浏览器**: Chrome 88+ 或 Chromium-based 浏览器
- **目标网站**: 所有支持CSS的网页，特别优化OpenClaw界面
- **权限**: 仅需活动标签页和存储权限

## 🔧 技术架构

- **Manifest V3** - 现代扩展规范
- **Content Scripts** - 安全的CSS注入
- **Chrome Storage API** - 设置持久化
- **MutationObserver** - 动态DOM监听

